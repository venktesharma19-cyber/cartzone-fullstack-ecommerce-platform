# LinkedIn Project Description — CartZone

## Project title

CartZone — Full Stack Shopping Platform

## Short LinkedIn description

Built CartZone, a full-stack AI-enhanced shopping platform with React, TypeScript, Redux Toolkit, Node.js, Express, PostgreSQL, Redis, Stripe, Docker, and GitHub Actions. Implemented natural-language product discovery, budget-aware AI recommendations, AI review summaries, seller AI insights, Redis-backed cart persistence, JWT auth with refresh-token rotation, Stripe checkout/webhooks, order tracking, seller inventory dashboard, admin management, automated tests, and AWS S3-ready product image uploads.

## Recruiter-friendly explanation

CartZone is an Amazon-inspired, AI-enhanced shopping platform that demonstrates production-style full-stack engineering. Users can browse products, ask an AI shopping assistant for recommendations, filter/search listings, add items to a Redis-persisted cart, checkout through Stripe/demo checkout, review products, and track order status. Product pages include AI review summaries, while sellers get AI merchandising insights for restocking, trust gaps, and best-seller promotion. Sellers can add products and manage inventory, while admins can manage users, orders, and product listings.

## Resume bullets

- Added an AI commerce layer with natural-language shopping prompts, intent extraction, budget-aware product ranking, recommendation reasons, and buyer follow-up prompts.
- Built AI review summaries that convert customer reviews into sentiment, highlights, and buyer-friendly product insights.
- Added seller AI insights for inventory risk, low-stock alerts, review gaps, and best-seller merchandising recommendations.
- Built a full-stack e-commerce platform using React, TypeScript, Redux Toolkit, Node.js, Express, PostgreSQL, Redis, and Docker.
- Implemented JWT authentication with refresh-token rotation, email verification flow, and role-based access for buyers, sellers, and admins.
- Designed PostgreSQL schema for products, categories, orders, order items, reviews, users, and refresh tokens.
- Built Redis-backed cart persistence using session IDs to preserve cart state across page refreshes.
- Integrated Stripe Checkout with webhook handling and added a local demo fallback for recruiter-friendly project testing.
- Added seller dashboard for product creation, inventory management, and sales metrics.
- Added admin panel to manage users, order statuses, and product visibility.
- Added Jest and Supertest coverage for API health, token logic, cart logic, and order total processing.
- Dockerized frontend, backend, PostgreSQL, and Redis; added GitHub Actions CI pipeline for type-checking, testing, and frontend builds.
